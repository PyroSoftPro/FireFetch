üîê Authentication & Private Content

Cookie Authentication

For downloading private, age-restricted, or premium content, you can upload browser cookies:

  1. Install the "Get cookies.txt LOCALLY" browser extension
  2. Visit the video site and log in to your account
  3. Click the extension icon and export cookies as .txt file
  4. Go to Settings ‚Üí Authentication and upload the cookie file
üîí Privacy: Cookie files are stored locally and never shared. They're only used to authenticate your downloads.

üèéÔ∏è Performance Optimization

Download Acceleration Settings

Concurrent Connections

Number of parallel download connections (1-32). More connections = faster downloads for large files.

Segments per Connection

Splits each connection into smaller segments (1-32). Helps with unreliable connections.

Segment Size

Size of each download chunk (512KB - 5MB). Larger segments = less overhead.

‚ö° Optimization Tips: For very fast connections, increase concurrent connections. For slower/unstable connections, reduce connections but increase segments.

‚ùó Troubleshooting

Common Issues & Solutions

üö´ "Authentication Required" Error

  • The video requires login to access
  • Upload browser cookies in Settings ‚Üí Authentication
  • Make sure you're logged in to the site in your browser

üîí "DRM Protected Content" Error

  • The content is protected by digital rights management
  • This cannot be downloaded legally
  • Try finding alternative sources or official download options

‚ö†Ô∏è Download Fails or Stops

  • Check your internet connection
  • Try reducing concurrent connections in settings
  • The source file may no longer be available
  • Enable automatic retry in queue settings

üêå Slow Download Speeds

  • Increase concurrent connections for faster internet
  • Check if the source server has rate limiting
  • Try downloading during off-peak hours
  • Consider using a different quality option
